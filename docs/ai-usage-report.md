# AI Usage Report – Assignment 2

## Tools Used
- ChatGPT

---

## How I Used AI

During the development of this assignment, I used AI mainly as a learning and support tool while implementing different features in the portfolio website.

- I used AI to generate some initial code suggestions and explanations for implementing the **project search feature**, which allows users to search for projects by keywords.
- I asked AI for guidance on how to implement **project filtering** using JavaScript and data attributes to show or hide project cards.
- I used AI to understand how to implement **active navigation highlighting while scrolling**, so that the current section in the page is highlighted in the navigation bar.
- I used AI to learn how to **fetch data from a public API** and display it dynamically on the website. This was used to display a developer quote using the Advice Slip API.
- I asked AI for suggestions on improving the **structure and readability of my CSS and JavaScript code**, including organizing styles into sections and adding comments.
- I used AI to help debug smaller issues such as **email validation patterns**, layout adjustments, and improving some user interface behaviors.
- Used AI to help structure and write parts of the project documentation, including the README file and technical documentation.

---

## Benefits

Using AI helped me better understand how different front-end features work.

- It helped me understand how **JavaScript interacts with the DOM**.
- It made it easier to learn how to implement interactive features such as **search, filtering, and API data fetching**.
- It helped me identify and fix coding errors more efficiently.
- It provided suggestions for improving the **organization and readability of my code**.

---

## Challenges

While AI provided useful suggestions, some challenges still required manual work.

- Some generated code needed to be **adjusted to fit my existing project structure**.
- Not all suggestions worked immediately, so I had to **test and modify the code** to ensure it worked correctly.
- Understanding how the generated code worked required **additional reading and experimentation**.

---

## Learning Outcomes

This assignment helped me improve several technical skills.

- I improved my understanding of **DOM manipulation and event handling in JavaScript**.
- I learned how to **integrate a public API using the fetch function** and display dynamic content.
- I gained experience implementing **interactive UI features such as filtering, search, and scroll-based navigation behavior**.
- I learned how to **review, understand, and adapt AI-generated suggestions instead of simply copying them**.

---

## Responsible Use

AI was used as a **support tool during development**, not as a replacement for my own work.

- All AI-generated suggestions were **reviewed, modified, and tested** before being included in the project.
- I ensured that I **understood how the code works** before adding it to the final implementation.
- The final structure and implementation of the project reflect **my own understanding and decisions**.
